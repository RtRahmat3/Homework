CSS Selector

The element Selector Select HTML elements based on the element name.

            
                Selector{
                    Property : Value;
                }
                Like:

                p{
                    text-align:Center;
                }

CSS Id Selector

The id selector select specific element with unique Name.

            
                #id{
                    text-align:Center
                }

CSS Class Selector

The Class selector select HTML elements with a specific class element

            
                .id{
                    text-align :Center
                }

Multiple Selector

The Class selector select HTML elements with a specific class element

            
                .id div #div{
                    text-align:center;
                    color:black;
                }
